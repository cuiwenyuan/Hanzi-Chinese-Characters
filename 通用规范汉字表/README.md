# 通用规范汉字表

本项目包含《通用规范汉字表》（2013年版）的完整Markdown版本，方便开发者、语言学习者和研究人员使用。

## 📋 项目简介

《通用规范汉字表》是中华人民共和国教育部和国家语言文字工作委员会于2013年发布的汉字规范标准，共收录汉字8105个。

## 📊 文档内容

### ✅ 完整收录
- **全部8105个汉字**，无任何遗漏

### ✅ 三级分类
| 级别 | 编号范围 | 汉字数量 | 说明 |
|------|----------|----------|------|
| 一级字表 | 1-3500 | 3500字 | 常用字，覆盖99.48%的使用频率 |
| 二级字表 | 3501-6500 | 3000字 | 次常用字 |
| 三级字表 | 6501-8105 | 1605字 | 通用字 |

### ✅ 表格字段
| 字段 | 说明 | 示例 |
|------|------|------|
| 编号 | 汉字在表中的序号 | 0001 |
| 汉字 | 规范汉字 | 一 |
| 拼音 | 汉语拼音（多音字用逗号分隔） | yī |
| 繁体 | 对应的繁体字（如有） | (一) |
| 异体 | 异体字（如有） | [壹] |

## 🎯 文档特点

- **Markdown表格格式**：清晰易读，支持GitHub、GitLab等平台直接渲染
- **三级分类**：按使用频率分级，方便快速查找
- **信息完整**：包含拼音、繁体、异体等完整信息
- **多种格式**：提供完整版和分块版两种形式

## 🚀 使用方式

### 方式一：使用完整版
打开 `通用规范汉字表_完整版.md`，包含所有8105个汉字，按级别分节。

**适用场景**：
- 需要完整查看所有汉字
- 进行全文搜索
- 打印或导出为PDF

### 方式二：使用分块版
进入 `markdown_output/` 目录，每个文件包含100个汉字（最后一个文件包含5个）。

**适用场景**：
- 分批处理汉字
- 按需加载，提高性能
- 集成到应用程序中

### 方式三：使用文本数据
打开 `通用规范汉字表_文本.txt`，原始TSV格式数据。

**数据格式**：
```
编号\t汉字\t拼音\t繁体\t异体
```

**适用场景**：
- 程序化读取和处理
- 导入数据库
- 自定义格式化输出

## 💡 应用场景

1. **中文学习**：查找汉字的拼音、繁体、异体字信息
2. **字体设计**：确保字体覆盖所有规范汉字
3. **输入法开发**：构建完整的汉字库
4. **自然语言处理**：中文分词、OCR识别等任务的字符集参考
5. **数据分析**：汉字使用频率统计和分析

## 🔧 技术实现

### 数据来源
由于原始PDF为扫描版，无法直接提取文本，本项目从jsdelivr CDN获取标准文本数据，确保数据的准确性和完整性。

### 生成脚本
`生成完整文档.py` 是Python脚本，用于：
- 读取TSV格式的文本数据
- 按级别（一级、二级、三级）分节
- 生成Markdown格式的表格

**运行方式**：
```bash
python 生成完整文档.py
```

## 📝 数据说明

### 拼音字段
- 单音字：直接显示拼音，如 `yī`
- 多音字：用逗号分隔，如 `gān, gàn`
- 无声调：拼音后面的数字表示声调（本项目省略声调数字）

### 繁体字段
- 格式：`(繁体字)` 或 `(繁体1|繁体2)`
- 示例：`(干|乾|幹)` 表示"干"有多个对应的繁体字
- 无繁体：字段为空

### 异体字段
- 格式：`[异体字]` 或 `[异体1|异体2]`
- 示例：`[|乹亁|榦]` 表示"干"的异体字
- 无异地：字段为空

## ⚠️ 注意事项

1. **拼音声调**：本数据省略了声调数字（1-4声），实际应用中可能需要补充
2. **繁体匹配**：部分汉字有多个繁体形式，使用时需注意上下文
3. **异体字**：异体字在现代中文中很少使用，主要用于古籍研究
